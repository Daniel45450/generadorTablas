# Notas

Argumentos para pasarle al programa

```

Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Toma la base de la tabla de multiplicar
                                                [number] [required] [default: 5]
  -l, --listar   Muestra la tabla en consola          [boolean] [default: false]
  -i, --inicio   Indica el comienzo de la tabla            [number] [default: 1]
  -f, --fin      Indica el final de la tabla              [number] [default: 10]

```
